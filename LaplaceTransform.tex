If time continues
\begin{equation*}
	\int_{0}^{\infty} \red{a(t)} x^t \dd t = \blue{A(x)}  \qc \text{收敛条件：} 0<x<1
\end{equation*}
根据恒等式
\begin{gather*}
	x = e^{\ln x}\\
	x^t = e^{t \ln x}
\end{gather*}
\(-s = \ln x < 0 \implies s>0\)
\begin{remark}
	为什么要令 \(-s = \ln x\)?

	因为 \(\ln x\) 自然地限制了 \(x > 0\)，只要再加一个条件\(\ln x < 0\) 就又限制了 \(x<1\)。
\end{remark}

\begin{equation*}

\end{equation*}

\end{itemize}
